# 2. Requisitos Funcionais

A Tabela 4 apresenta os Requisitos Funcionais elicitados para o sistema BeautyFlow.

| ID | Requisito Funcional | Descrição | Prioridade |
|----|----------------------|-----------|------------|
| RF01 | Cadastro de Clientes | Permitir cadastrar clientes com nome, telefone e e-mail. | Alta |
| RF02 | Editar Clientes | Permitir editar informações de clientes cadastrados. | Média |
| RF03 | Cadastro de Serviços | Permitir cadastrar serviços com valor e duração. | Alta |
| RF04 | Agendamento | Permitir agendar clientes por data e horário. | Alta |
| RF05 | Visualização da Agenda | Exibir atendimentos organizados na agenda. | Alta |
| RF06 | Controle de Estoque | Registrar entrada e saída de produtos. | Média |
| RF07 | Fluxo de Caixa | Registrar entradas e saídas financeiras. | Alta |
| RF08 | Relatórios | Gerar relatórios gerenciais. | Média |
